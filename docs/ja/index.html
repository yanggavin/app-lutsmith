よくある質問

ソースメディアはアップロードされますか？

いいえ。LUT Smith は写真、RAW 画像、選択した動画フレームを端末上で処理します。ソースメディアは LUT Smith のサーバーにアップロードされません。

アカウントは必要ですか？

ローカル LUT プロジェクトにアカウントは不要です。ライブラリからインポートを選んだときだけ写真へのアクセスを求めます。

サブスクリプションはどう機能しますか？

LUT Smith Pro は App Store で販売されます。購入とサブスクリプション状態は Apple と RevenueCat が管理します。設定から購入を復元できます。

.cube LUT には何が含まれますか？

標準 .cube LUT はグローバルな色変換を保存します。グレイン、マスク、ぼかし、グローなどプレビュー専用の効果は、グローバルな色変化として表現できない限り含まれません。

互換性をどう確認しますか？

17× のアイデンティティ LUT とスタイル付き LUT を書き出し、両方を先アプリに読み込みます。入出力プロファイルが一致していれば、アイデンティティは無変更、スタイル付きはプレビューと一致するはずです。
